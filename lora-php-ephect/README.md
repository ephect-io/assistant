# LoRA PHP + Ephect Code Trainer (Dataset Generator Edition)

This repository fine-tunes a code LLM for **PHP + Ephect framework** using **QLoRA**.

## New in v2
- Automatic dataset generation from the Ephect framework source
- AST-free, token-safe PHP extraction (classes, methods, docblocks)
- Instruction synthesis specialized for Ephect patterns

You must clone the Ephect framework locally.
